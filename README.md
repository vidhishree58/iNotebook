# 📝 iNotebook

A full-stack MERN Notes Application that allows users to securely create, update, and delete personal notes using JWT Authentication.

## 🚀 Features

- User Signup & Login
- JWT Authentication
- Create Notes
- Update Notes
- Delete Notes
- Secure Password Hashing (bcrypt)
- Responsive UI
- MongoDB Database

## 🛠️ Tech Stack

### Frontend
- React.js
- React Router DOM
- Bootstrap 5
- Context API

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs
